# ceraxus
  
Get and summarize iOS app reviews.
  

## Docker
  
### Version
  
```bash
> docker --version

> docker-compose --version
```

### Build (docker-compose build)
  
```bash
> make dc-build
```
  
### Run
  
```bash
> make run
```
